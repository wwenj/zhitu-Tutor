## 技术选型

- 移动端

- **前端：** vue-cli，vue-router，mint.ui，axios，es6，

- **后端：** PHP

## 功能实现

1. **首页：** 轮播，分类，个人信息，教师列表
2. **详情页：** 教师简介，预约，收藏
3. **分类页：** 年级，学科，教师类型分类
4. **登录注册页：** 个人登录，快捷登录
5. **个人信息页：** 教师，家长两种
6. **个人信息编辑页：** 教师，家长两种
7. **收藏页：** 查看家长收藏教师列表
8. **其他页面：** 我的课程（教师），我的试讲（教师），我的预约（家长），网站介绍，客服电话

## Build Setup

``` bash
# 下载依赖
npm install

# 打开端口 localhost:8001
npm run dev

# build for production with minification
npm run build
```
