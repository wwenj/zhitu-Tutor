
## 教师精简结构

**简要描述：** 
- 教师精简结构，列表使用

**返回示例**

``` 
        {
          "id" : 1,
          "name" : "",   
          "image" : "",   
          "subject" : [
          	  {
          	  	"id" :1,
          	  	"lebel" : "数学",
          	  	"sort"  : 0
          	  }
          ],     
          "grade" : [
          	  {
          	  	"id" :1,
          	  	"lebel" : "数学",
          	  	"sort
          	  }
          ],   
          "extend_tag" : [
          	  {
          	  	"id" :1,
          	  	"lebel" : "已认证",
          	  	"sort"  : 0
          	  }，
          	  {
          	  	"id" :1,
          	  	"lebel" : "大学生",
          	  	"sort"  : 0
          	  }
          ],
          "school" : "",
          "source" : 1,   
          "is_confirm" : 0,  
          "is_star" : 0,  
          "teach_time" : 10, 
          "teach_time" : 10,        
        }
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|source |int   |来源类型 1:大学生 2:编制内教师 3:职业教师  |
|extend_tag |arr   |拓展tag  |
